Chrysus is a complex economy plugin for bukkit based Minecraft servers.

Features: 
-player run companies                                                                                                                                                                                                                                                                                                                                 
-dynamic pricing                                                                                                        
-buying selling of player's goods                                                                                       
-economic safeguards                                                                                                          
-WorldGuard integration                                           
-SQL storage
